# cdi-sample

Contexts and Dependency Injection (CDI) sample code with [Weld](http://weld.cdi-spec.org/)

## How to build/run

```bash
$ gradle run
```

```bash
$ gradle test
```

## How to build executable jar and run it

```bash
$ gradle shadowJar
$ java -jar build/libs/cdi-sample-all.jar
```
